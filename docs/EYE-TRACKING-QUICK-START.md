**5분 만에 시작하기!**

## 🚀 바로 테스트하기

### 1단계: Live Server 실행
```bash
# VSCode/Cursor에서
# index.html 우클릭 → "Open with Live Server"
```

### 2단계: 브라우저에서 확인
웹 페이지가 자동으로 열립니다.

### 3단계: Eye Tracking 활성화
1. 우측 툴박스 스크롤
2. **"♿ Accessibility (PoC)"** 섹션 찾기
3. **"👁️ Eye Tracking"** 버튼 클릭
4. 웹캠 권한 허용

### 4단계: 보정하기
화면의 여러 지점을 10회 이상 클릭하세요!
- 클릭할 때마다 녹색 원 애니메이션 표시
- 클릭할수록 시선 커서 정확도 향상

## 🚧 아직 없는 것

- ⏳ 시선으로 그림 그리기 (Dwell)
- ⏳ 9-point 보정 화면
- ⏳ 눈 제스처 (깜빡임, 윙크)

---

## 💡 팁

### 정확도 높이는 방법
1. **밝은 조명**: 얼굴에 그림자가 없도록
2. **정면 응시**: 모니터 중앙 정면에서
3. **거리**: 40-60cm 떨어지기
4. **보정**: 10회 이상 클릭 (많을수록 좋음)

### 문제 해결
- **커서가 안 보여요**: 콘솔(F12) 확인, 웹캠 권한 확인
- **정확도가 낮아요**: 더 많이 클릭해서 보정하세요
- **느려요**: 다른 탭 닫고, 조명 밝히기
